Vite & Gourmand — Application de commande en ligne (ECF DWWM)
URL production : https://ecf-vite-et-gourmand-self.vercel.app
Dépôt : https://github.com/OFF-8/ecf-vite-et-gourmand

1. Présentation

Application web permettant aux clients de consulter des menus traiteur, passer commande (retrait ou livraison), suivre leur commande et laisser un avis. Les employés gèrent le stock, les commandes et les contenus. L'administrateur gère les employés et consulte les statistiques.

3. Parcours client

  1. Accueil — découvrir l'offre et accéder aux menus.
  2. Nos menus — filtrer par prix, thème, régime ; ouvrir un détail.
  3. Inscription / Connexion — créer un compte ou se connecter.
  4. Commander — depuis un menu, renseigner nombre de personnes, date, retrait ou livraison.
  5. Mon espace — suivre les commandes, modifier le profil, laisser un avis après livraison.
  6. Mot de passe oublié — demander un lien de réinitialisation (email si SMTP configuré).

4. Parcours employé

  1. Se connecter avec Julie.
  2. Commandes — changer les statuts (acceptée → préparation → livraison → livrée → retour matériel → terminée / annulée).
  3. Menus / Plats / Horaires — maintenir le catalogue et les horaires d'ouverture.
  4. Avis — modérer les avis clients.

5. Parcours administrateur

  1. Se connecter avec José.
  2. Employés — créer / activer / désactiver des comptes employés.
  3. Statistiques — consulter les graphiques (données agrégées, MongoDB si configuré).
